Aurora is a modern, lightweight, and beautiful AI chat client designed for Windows. It leverages the Fluent UI design system to feel right at home on your desktop.

## ‚ú® Features

- **Fluent Design**: Native Windows 11 aesthetics with Mica effect support.
- **Multi-Provider Support**: Seamlessly switch between OpenAI, Gemini, Claude, and custom endpoints (OpenAI-compatible).
- **Session Management**: Organized chat history with local storage persistence.
- **Markdown Support**: Rich text rendering including code blocks, tables, and math equations.
- **Reasoning Display**: Support for thinking/reasoning models (e.g., DeepSeek R1).
- **Clipboard & Image Support**: Optimized for quick pasting (Win+V support) and drag-and-drop.
